Measuring the User Experience

The Analytics of UX

This is a simple demo web app offering a place to practice active analytics. What you'll see here is:

  • A few UI elements serving as interaction points.
  • JavaScript functions attached to change and click DOM events.
  • Use of Google Analytics to actively track events.

Using these concepts you'll build webapps, measure exactly how they're used, and learn what parts are used most often by your customers.
