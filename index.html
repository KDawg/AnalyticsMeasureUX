Welcome to this Sample App

This is a simple demo webapp offering a place to practice active analytics. What you'll see here is:

  • A few UI elements serving as interaction points.
  • JavaScript functions attached to change and click DOM events.
  • Use of Google Analytics to actively track events with analytics data.

Using these concepts you'll make webapps, measure exactly how they're used, and better understand what parts are the most useful for your customers.
